> CSS 우선순위

- 같은 레벨에서는 나중에 선언된 선택자가 우선! => Cascading

- 상위객체의 속성이 높더라도 상속으로 받은 객체는 우선순위가 마지막

- CSS 우선순위는 점수의 형태로 계산되는데, 더 정확히 명시(자손, 형제 등의 사용)될수록 점수가 더해져 우선순위가 높아짐.

- justify-content-center, align-items-center 잘 보기
