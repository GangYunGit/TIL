# 델타 탐색

- 이차원 리스트의 인덱스(좌표)의 조작을 통해서 상하좌우 탐색을 한다. 이때 행과 열의 변량인 -1, +1을 `델타(delta)`값이라 한다.

- `방향을 바꾸`는건 `델타값의 인덱스를 조작`하여 수행한다!

- 상하좌우로 이동 후 범위를 벗어나지 않는지 확인 및 갱신

  ```python

  for x in range(4):
      for y in range(4):
          for k in range(4)
              nx = x + dx[k]
              ny = y + dy[k]
              if 0 <= nx < 3 and 0 <= ny < 3
                  x = nx
                  y = ny
  ```

- 참고) 상하좌우 + 대각선 방향의 델타값

  ```python
  dx = [-1, 1, 0, 0, 1 ,-1, 1, -1]  # ← → ↑ ↓ ↘ ↙ ↘ ↖
  dy = [0, 0, -1, 1, 1, 1,-1, -1 ]  # ← → ↑ ↓ ↘ ↙ ↘ ↖
  ```
